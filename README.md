#Coding Challenge #4


## **Yêu cầu hệ thống**
- Node.js >= 22.17
- Yarn hoặc npm

## **Cài đặt**
Clone dự án về máy:
```bash
git clone https://github.com/thaibinhtes/be-skipli
cd be-skipli
copy .env.example .env
```

## **Cài đặt dependencies**
```bash
yarn install
# hoặc
npm install
```

#Chạy dự án (Development)
```bash
yarn start
# hoặc
npm run start
```
